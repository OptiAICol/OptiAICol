---

Diseño, construyo y dejo corriendo sistemas de IA, datos y automatización end-to-end para empresas en LATAM. Trabajo con el stack completo y elijo la herramienta y el modelo de IA adecuados para cada caso según costo, latencia y resultado.

---

### Lo que construyo

🤖 Agentes y automatizaciones en producción, orquestados en n8n e integrados a APIs de negocio  
🗄️ Arquitectura de datos con Supabase, PostgreSQL y MySQL — seguridad real (RLS, RPC)  
📱 Apps móviles end-to-end con React Native + Expo, publicadas en tienda  
🔗 Integración de LLMs según el caso (costo, latencia, tarea), visión artificial y RAG
